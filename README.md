## ✨ Features

- **Cinematic Hero Section** — Full-viewport portrait with GSAP entrance animations and scroll parallax
- **Animated Role Text** — Smooth infinite vertical scroll cycling through developer roles
- **Scroll-Driven Animations** — GSAP ScrollTrigger powers progressive content reveals on scroll
- **Smooth Scrolling** — Lenis-powered buttery smooth scroll experience
- **Custom Cursor** — Framer Motion animated cursor with hover state detection
- **Dark Luxury Aesthetic** — Deep `#0B0B0B` background with orange `#FF6B35` accent system
- **Glassmorphism Navigation** — Frosted-glass fixed header with social links
- **Project Showcase** — Three featured projects with image hover effects and GitHub links
- **Responsive Design** — Mobile-first layout that adapts seamlessly across all screen sizes

## 🚀 Getting Started

### Prerequisites
- Node.js v18+
- npm

### Installation

```bash
# Clone the repository
git clone https://github.com/sai-krishna26/sai-krishna26.github.io.git
cd sai-krishna26.github.io

# Install dependencies
npm install

# Start the development server
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

---

## 📦 Build & Deploy

```bash
# Build for production
npm run build

# Deploy to GitHub Pages (one command)
npm run deploy
```

The `deploy` script runs `vite build` then pushes the `dist/` folder to the `gh-pages` branch automatically.
